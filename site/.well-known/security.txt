# What is in scope
#
# The gateway itself (this repository), the hosted demo at
# mcp.cortex-gateway.dev, and its authorization server at
# auth.cortex-gateway.dev. The demo runs a deliberately trivial backend; a
# finding there only matters if it reaches past that backend.
#
# Known and documented, so please do not report as new: the MCP->backend
# adapter is beta and has never run against a real third-party provider; the
# tool-integrity baseline is signed but its trust chain stops at the operator,
# not the backend author. Both are stated in docs/security.md.
